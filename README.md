Controle e monitoramento de niveis de reservatório de água

Este projeto consiste em um sistema de automação para controle e monitoramento de níveis de água em reservatórios. O objetivo é evitar transbordamentos, prevenir danos às bombas hidráulicas (operação a seco) e otimizar o consumo de energia através de uma lógica de controle precisa.

Funcionalidades:
Monitoramento em Tempo Real: Leitura contínua dos níveis de água via sensores.
Automação de Bomba: Acionamento automático baseado em pontos críticos (níveis mínimo e máximo).
Segurança: Proteção contra funcionamento a seco e detecção de anomalias (vazamentos).
